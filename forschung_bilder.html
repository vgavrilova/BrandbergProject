Bilder aus der Forschung

Während der Forschung wurden ganz viele Fotos von Brandberg gesammelt. Sie werden Ihnen hier zur Verfügung gestellt, damit Sie noch mehr vom Brandbergmassiv erfahren könnten. Die Bilder können Sie einfach nach Schluchten sortieren, indem Sie die für Sie relevante Schlucht im Select-Box auswählen.

Die einzelnen Fotos wurden aus der IANUS-Datenbank entnommen. Die ausführlichen Informationen über jede Schlucht können Sie unter diesem Link bekommen.
